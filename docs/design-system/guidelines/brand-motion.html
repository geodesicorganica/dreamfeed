Rules
All transitions use --ease-crisp. No spring physics, no atmospheric bounce, no perceived smoothness for its own sake.
Local acknowledgement under 100ms. The operator must see state change before they lift their finger — --dur-instant: 80ms is the floor.
All animations gated on @media (prefers-reduced-motion: no-preference). The static end-state is always the readable, complete state.
Forbidden: loading skeletons that shift layout, infinite decorative loops, parallax, scroll animations, entry reveals.
Flash coding — restricted to mission-critical alerts only
